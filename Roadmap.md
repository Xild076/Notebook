## Potential Features

### High Impact
- [ ] Quick Switcher (Ctrl+O) — Fast fuzzy search to open any file
- [ ] Command Palette (Ctrl+P) — Central hub for all actions
- [ ] Backlinks Panel — Show "Files that link to this file"
- [ ] File Context Menu — Right-click for Rename, Delete, Duplicate, Move
- [ ] Hover Preview — Popup preview when hovering over links
- [ ] Daily Notes — One-click to create/open today's note

### Medium Impact
- [ ] Outline/TOC Panel — Document headings for quick navigation
- [ ] Tags Support — Parse #tags, tag search, tag cloud
- [ ] Recent Files — Quick access to recently opened files
- [ ] Starred/Pinned Files — Mark important files
- [ ] Status Bar — Word count, character count, line count
- [ ] Templates — Create new files from templates

### Quick Wins
- [ ] File Type Icons — Different icons for .md, .excalidraw, .kanban in explorer
- [ ] Keyboard Shortcuts Modal (Ctrl+?)
- [ ] Auto-save indicator — Show "Saving..." / "Saved" status
- [ ] Drag files into editor — Drop to insert embed or link
